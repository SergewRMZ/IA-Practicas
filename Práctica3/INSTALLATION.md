# 📦 GUÍA COMPLETA DE INSTALACIÓN

## Requisitos del Sistema

- **Python:** 3.8 o superior
- **Sistema Operativo:** Windows, Mac o Linux
- **Navegador:** Chrome, Firefox, Safari o Edge (cualquier navegador moderno)
- **Espacio en disco:** ~50MB
- **Conexión a internet:** Para descargar paquetes (durante instalación)

## Instalación Paso a Paso

### OPCIÓN 1: Instalación Automática (RECOMENDADO)

#### Windows (Símbolo del Sistema o PowerShell):
```bash
# 1. Ve al directorio del proyecto
cd ruta\a\tic-tac-toe-4x4

# 2. Instala dependencias
pip install -r requirements.txt

# 3. Ejecuta el servidor
python tictactoe_server.py

# Verás:
# * Running on http://127.0.0.1:5000 (Press CTRL+C to quit)
```
